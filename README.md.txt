# Live AI Voice Assistant

This is a basic web-based Live AI Voice Assistant project created using HTML and JavaScript.

## Features
- Simple UI built using HTML
- Voice interaction using browser APIs
- JavaScript handles logic and interaction

## Tech Stack
- HTML
- JavaScript

## Running the Project

This repository does not include an API key for security reasons.

To test the live AI functionality:
1. Generate a Google AI Studio API key
2. Replace "YOUR_API_KEY_HERE" in script.js with your own key
3. Run index.html locally
